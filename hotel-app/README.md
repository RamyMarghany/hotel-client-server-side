 This project get data from json server app [hotel-app-api](https://github.com/ramy-medhat/hotel-app-api).

  ### `setup the application`
- npm install

 ### `compiles data from json server app`
- npm run serve

### `build and compiles for production`
- npm run build

### `run test cases`
- npm run test

### `run the application`
- npm start
